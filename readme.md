## Getting Started

1. **Clone the repository**
   ```bash

   ```

2. **(Optional) Create a virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Run the app**
   ```bash
   python main.py
   ```
